# RaffleTools
一个可以用于抽奖的沙雕小工具（由工口发动机强力驱动）

使用方法：下载Release中的打包exe执行即可。

## Development

1. Clone this repository:

   ```bash
   git clone https://github.com/JXNUACS/RaffleTools
   ```

2. Install pip packages:

   ```bash
   pip install -r requirements.txt
   ```
   
3. Modify code in `main.py`

4. Build

   ```
   python build.py
   ```

5. Enjoy it!
